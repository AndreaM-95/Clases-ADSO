CSS

¿Qué es CSS?

Hojas de estilo en cascada (CSS) es un lenguaje de hojas de estilo utilizado para describir la presentación de un documento escrito en HTML o XML..

¿Para qué sirve CSS?

CSS se utiliza para definir el diseño y la apariencia de las páginas web, incluyendo colores, fuentes, espaciado y disposición de los elementos.

¿Cómo se usa CSS?

CSS se puede aplicar a los documentos HTML de tres maneras: en línea, en la sección <head> del documento, o mediante un archivo externo.
